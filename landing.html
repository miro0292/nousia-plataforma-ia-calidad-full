PROMECIA

Plataforma inteligente que analiza métricas de calidad de código, predice el esfuerzo para mejorar un servicio y proyecta el impacto de implementar buenas prácticas de ingeniería.

Ver Características

¿Qué hace PROMECIA?

Cálculo de Riesgo Técnico

Analiza densidad de code smells, issues críticos, duplicación, complejidad cognitiva y cobertura de pruebas.

Estimación Inteligente

Modelo basado en XGBoost que predice horas de esfuerzo para alcanzar un nivel de calidad objetivo.

Simulación SonarQube

Conexión simulada con Sonar para cargar métricas reales de diferentes servicios.

Plan de Mejora Automático

Sugiere distribución del esfuerzo: refactor, pruebas, seguridad y gestión.

Estados A–E

Muestra el nivel actual y proyectado del servicio (A–E), mostrando la mejora al aplicar las buenas prácticas.

Exportación PDF

Reporte ejecutivo listo para entregar a arquitectura, calidad o management.
